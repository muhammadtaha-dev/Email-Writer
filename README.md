Voice-to-Professional Email Generator


An AI-powered application that converts voice input (Urdu, Roman Urdu, or English) into professional business emails in English.  
Perfect for professionals, freelancers, and students who want to quickly turn spoken thoughts into well-structured emails.



Features

Voice input support (Urdu, English, Roman Urdu)
AI-powered speech-to-text processing
Automatic professional email generation
Tone customization (formal, professional, friendly)
Fast and minimal workflow
Multi-language understanding


How It Works

1. User records voice input  
2. Speech is converted into text (STT model / API)  
3. AI understands intent and context  
4. A professional email is generated in English  
5. Output is displayed or copied
